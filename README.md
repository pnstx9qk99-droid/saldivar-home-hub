# 🏠 Cleaning Routine Tracker

A beautiful, interactive checklist web application to manage your entire cleaning routine—from daily tasks to yearly maintenance.

## ✨ Features

- ✅ **Checkbox tracking** for all tasks (daily, weekly, monthly, seasonal, yearly)
- 💾 **Auto-save** to browser's local storage
- 📱 **Fully responsive** design for mobile, tablet, and desktop
- 🎨 **Color-coded callout blocks** for important information
- 📊 **Organized sections** with emoji headers
- 🔔 **Last updated timestamp** to track your progress
- 🌐 **Works offline** with PWA support
- 🎯 **Quick actions**: "Today" button to reset daily tasks, "Clear All" for completed tasks

## 📋 Included Routine

### 🌅 Daily — Morning
- Make bed
- Empty dishwasher
- Wipe bathroom counter & sink
- Clear & wipe kitchen counters
- Start laundry (rotation schedule)
- 5-min living room tidy

### 🌙 Daily — Evening Reset
- Move laundry to dryer or fold
- Wash/load dishes
- Wipe kitchen counters & stovetop
- Sweep kitchen floor
- Take out trash if full
- Final living room reset

### 📅 Weekly Tasks
Monday through Saturday, each with specific focus areas and rotating laundry schedule

### 🗓️ Monthly Tasks
30 daily maintenance tasks throughout the month

### 🌿 Seasonal Tasks
Tasks organized by quarter and specific months

### 🌞 Yearly — Summer Tasks
Annual deep cleaning and maintenance

### ⚠️ Safety Information
Important warnings about mixing cleaning products

### 🛒 Cleaning Cart Inventory
Complete list of cleaning supplies and their organization

## 🚀 Getting Started

1. Open `index.html` in your browser
2. Check off tasks as you complete them
3. Your progress is automatically saved
4. Use "Today" button to reset daily tasks
5. Use "Clear All" to clear all completed tasks

## 💾 Local Storage

Your checked tasks are automatically saved to browser storage. They persist even when you close and reopen the page.

## 📱 Mobile App

This site works as a Progressive Web App (PWA):
- Add to home screen on mobile devices
- Works offline
- Fast loading times

## 🛠️ Technology Stack

- HTML5
- CSS3 (with custom properties)
- Vanilla JavaScript
- LocalStorage API
- Service Workers (PWA)

## 📝 Customization

To customize the tasks, edit `data/tasks.js` and modify the task arrays and objects.
